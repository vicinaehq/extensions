# Snapper Snapshots Changelog

### Friday, July 17, 2026

- Initial release.
- Browse Snapper configs and snapshots with a decoded, readable changed-files view and
  on-demand per-file diffs.
- Instant local file search inside any snapshot.
- Create snapshots, and a one-time access setup so browsing works without a password prompt.
- Open a changed file's snapshot or current version, or reveal it in the file manager.
