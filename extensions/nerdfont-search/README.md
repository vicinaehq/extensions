# NerdFonts

Search Nerd Font icons in Vicinae and copy glyph values in multiple formats.

## What It Does

- Search icons by name, identifier, and fuzzy token matching.
- Copy any result as:
  - glyph
  - Nerd Font name
  - identifier
  - Unicode codepoint
  - HTML entity
- Keep a local recent-copies list (up to 24 icons).

## Commands

| Command | Description | Mode |
| --- | --- | --- |
| `nerdfonts-search` | Search Nerd Font icons and copy glyph values | View |

## Preferences

This extension currently has no user-configurable preferences.

## Requirements

- Node.js 20+

## Development

```bash
npm install
npm run dev
```

`predev` runs index generation automatically.

## Build

```bash
npm run build
```

`prebuild` runs index generation automatically.

## Tests

```bash
npm test
```

Tests run with Node's test runner and `tsx`:

```bash
node --import tsx --test test/*.test.ts
```

## Search Index Assets

The extension uses generated assets in `assets/`:

- `assets/glyphnames.json`
- `assets/icon-index.json`

They are generated by `scripts/build-index.js` during `predev`, `prebuild`, and `pretest`.

To force-refresh from upstream Nerd Fonts data:

```bash
npm run refresh:index
```

## Notes

- Icon pack filtering is currently behind a feature flag and disabled (`ENABLE_PACK_FILTER = false`).
