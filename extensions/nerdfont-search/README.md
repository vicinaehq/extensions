# NerdFonts

Search Nerd Font glyphs, preview them in a grid, and copy the glyph, Nerd Font name, identifier, codepoint, or HTML entity.

## Development

```bash
npm install
npm run dev
```

`predev` builds the search index from `assets/glyphnames.json` when it already exists and only downloads when the cache is missing.
Use `npm run refresh:index` to force-refresh `glyphnames.json` from Nerd Fonts upstream.

## Build

```bash
npm run build
```

## Tests

```bash
npm test
```

`assets/glyphnames.json` and `assets/icon-index.json` are generated by `scripts/build-index.js` during `prebuild`, `predev`, and `pretest`. They are intentionally ignored by git.
