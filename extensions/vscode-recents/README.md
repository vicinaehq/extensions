# VSCode recents extension <img src="assets/extension_icon.png" width="40" height="40" alt="VSCode recents extension icon" style="vertical-align: middle; ">

This extension provides a command to quickly access and open recently edited Visual Studio Code projects directly from Vicinae.

## VSCode Flavour

The extension supports the following VSCode flavours:
- Visual Studio Code
- Visual Studio Code (Insiders)
- VSCodium
- Cursor
- Antigravity

## Window Preference

The extension supports the following window preferences:
- New Window
- Reuse Window
- Default (files in existing, folders in new window)

## Database Backend

The extension supports the following database backends:
- sqlite3 (faster, requires system binary)
- sql.js (bundled, works everywhere)

## Usage

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the extension in development mode:
   ```bash
   npm run dev
   ```
3. Build the production bundle:
   ```bash
   npm run build
   ```
