# Vicinae-Mail

A minimal mail client for [Vicinae](https://github.com/vicinaehq/vicinae) using [imapflow](https://www.npmjs.com/package/imapflow).

besides browsing your inbox and reading your mail, the mail client has the following niceties:
- connect to multiple accounts
- format html mail to markdown (uses [turndown](https://www.npmjs.com/package/turndown) under the hood) 
- opening mails for replying using your full-fledged mail client
