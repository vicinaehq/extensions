# Déjà Dup Backups Changelog

### Friday, July 17, 2026

- Initial release.
- Browse snapshots and restore individual files or folders.
- Fast, offline file search backed by a one-time local index (with optional background indexing).
- Back Up Now and Backup Status commands.
- Backends: local folder, removable drive, network server (SMB/SFTP/WebDAV/Nextcloud),
  Google Drive, OneDrive, and rclone remotes.
- Native, Snap and (best-effort) Flatpak installations, with manual overrides in preferences.
